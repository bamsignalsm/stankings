# Stankings HQ v1.0.0 — Initial Production Release

**Tag:** `v1.0.0`  
**Date:** 2026-07-04  
**Commit:** production build `f1d6c51` (and subsequent maintenance commits)

## Summary

First production release of **Stankings HQ** — the institutional headquarters and shared platform for the Stankings ecosystem.

## Included

- Public corporate website (About, Companies, Leadership, Constitution, Library, Careers, Press, Media, Partners, Investors, Foundation, Institute)
- Central Trust, Security, Legal, Support, Compliance, and Status centers
- Shared services registries (`src/lib/shared`)
- Design system (`src/lib/design-system`, `/design-system`)
- Founder War Room / Launch Command Center (`/energy/launch`)
- Docker-first Coolify packaging (Node 22 standalone)
- Health and readiness endpoints

## Not included (by design)

- Product runtimes for BamSignal, Yike, or BayRight
- Shared databases, authentication, or payments across products
- Fabricated uptime metrics or marketing claims

## Maintenance policy

HQ is in **Operational Maintenance**. Allowed work:

- Company additions (registry)
- Legal / policy updates
- Monitoring and security patches
- Dependency updates

No new product features on HQ while Stage 2 (BamSignal) is active.
