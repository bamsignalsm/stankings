# 12. Employee Experience

```
Careers → Passport (create if missing) → Apply → Applicant Dashboard
  → Interview → Offer → Accept → Workforce capability on same Passport
  → /skl Welcome Checklist → Role Dashboard
```

Workers use `/skl` (not Energy). Passport login is unified; routing chooses dashboard by capabilities.

## Welcome checklist (default)

1. Complete profile  
2. Read company knowledge base intro  
3. Acknowledge code of conduct  
4. Open role queue  
5. Message manager (if assigned)
